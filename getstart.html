2. 說明

請 import addressing-1.6.2 以及 IHE-Technical-Frameworks-actors 至 java 專案，並可在 main function 內填入以下範例程式碼。

3. 基本範例

以下範例展示三個交易：上傳、查詢、調閱。

  • 00.config 為 endpoint 以及 patient id 設定。
  • 01.pnr 為 Provide And Register DocumentSet 交易，Doc Src 上傳文件/目錄等功能。
  • 02.sq 為 Registry Stored Query 交易，Doc Consumer 查詢索引功能。使用 GetDocument 操作，利用 documentId 查詢文件之索引。
  • 03.parse 為解析索引，擷取出 Repository Id 以及文件的 UniqueId。
  • 04.ret 為 Retrieve DocumentSet 交易，Doc Consumer 調閱文件功能。依據上述擷取的資訊調閱文件具體的內容。
