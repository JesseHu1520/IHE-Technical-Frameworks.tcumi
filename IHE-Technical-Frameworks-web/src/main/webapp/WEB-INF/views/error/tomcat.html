Step 01. 前置作業

  • 設定 Java 環境變數)
  • 下載： Tomcat 7
  • 安裝：以 64 位元舉例, 解壓縮 apache-tomcat-6.0.44-windows-x64.zip 即可

Step 02. 啟動

  • 到目錄 bin/ 執行 startup.bat啟動成功最後會出現, "資訊; Server startup in xxxx ms "
  • 請至瀏覽器, 輸入 "http://localhost:8080/" 網址, 若有出現一隻貓的網頁則成功執行 Tomcat Server

Step 03. 發布網站

  • 將 .war 檔案，複製到 webapps
  • 重新啟動 tomcat
