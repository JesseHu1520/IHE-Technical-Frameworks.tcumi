Dev. Environment - Java SE Development Kit 7 (JDK)



Java 安裝



Step 01. 下載&安裝

  • Java SE Development Kit 7 (JDK)
  • Java Runtime Environment ：Java 程式在執行的虛擬機器
  • Java Development Kit：開發 Java 程式時需要的開發工具, 包含函式庫等


Step 02. 設定環境變數

  • (1) "我的電腦" 上點擊"右鍵"
  • (2) "內容"
  • (3) 左邊的進階系統設定
  • (4) 在 "進階" 標籤點選 "環境變數"
  • (5) 新增兩個 "系統變數":
  • (6-1) JAVA_HOME:依照你的 "JDK" 資料夾路徑放置 "變數值", 變數名稱為 " JAVA_HOME", 點選確定
  • (6-2) JRE_HOME:依照你的 "JRE" 資料夾路徑放置 "變數值", 變數名稱為 "JRE_HOME" , 點選確定
  • (6-3) 編輯 "系統變數" Path :
  • 找到 Path 點選 "編輯", 變數值原有的內容不刪除更動, 將游標移動到最後貼上 ";%JAVA_HOME%\bin;%JRE_HOME%\bin;"
  • 以分號 ";" 區隔分別為 , (6-3-1) %JAVA_HOME%\bin; , (6-3-2) %JRE_HOME %\bin;
  • 完成後依序點選確定
